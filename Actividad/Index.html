Actividades de JavaScript

Actividad 1: IF anidados - Clasificación de edades

Verifica la categoría de edad ingresada en la consola del navegador.

Actividad 2: FOR - Tabla de multiplicar

Genera la tabla de multiplicar de un número utilizando un bucle FOR.

Actividad 3: IF + FOR - Tablas pares o impares

Muestra las tablas de multiplicar solo de números pares o impares según selección.

Para verificar los resultados, abre la consola del navegador.
